CSS Attribute Selectors

What is an Attribute Selector?

Attribute selectors select HTML elements based on having a specific attribute or an attribute with a specified value

Types of Attribute Selectors

[attribute] Selector

Applies the style rule to an element if the given attribute exists

[title]{ color:blue } Styles all elements that have a title attribute with the color blue

example[attribute] Selector

Applies the style to all 'example' elements that have a given attribute

a[title]{ color:green } All '' elements with a title attribute will be styled green
